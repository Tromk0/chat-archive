### 版本功能整理與演進

---

#### **v1.0 (基礎版)**
- **核心功能**：實現最基本的對話總結歸檔。
- **主要操作**：
  - 在文字框貼上 LLM 生成的對話總結，點擊「解析並預覽」。
  - 手動或自動解析出「日期、分類、主題、關鍵詞、摘要、模式標記」。
  - 確認後保存至瀏覽器的 `localStorage`。
- **列表與檢視**：
  - 以卡片清單顯示所有記錄，支援按「分類」與「日期」篩選。
  - 點擊卡片可彈出 Modal 檢視完整原始內容。
- **數據管理**：
  - 可「導出 JSON」備份，或「導入 JSON」還原數據。
  - 提供「清空全部數據」的危險操作。
- **儲存機制**：僅使用 `localStorage`。

---

#### **v2.0 (體驗與提醒增強)**
- **新增功能**：
  - **深色模式**：支援手動切換，並跟隨系統偏好。
  - **備份提醒系統**：
    - 啟用後，若超過 7 天未備份，會在頁面頂部顯示提醒橫幅。
    - 記錄「上次備份日期」，並在設定頁面顯示。
  - **設定頁面優化**：將「深色模式」、「備份提醒」整合至設定頁的開關組件。
- **儲存機制**：仍僅使用 `localStorage`，但新增備份相關的儲存鍵。

---

#### **v3.0 (搜尋與生產力提升)**
- **新增功能**：
  - **全文搜索切換**：列表頁新增「全文搜索」開關，可選擇是否搜尋原始對話內容。
  - **搜尋關鍵字高亮**：在卡片標題與摘要中，高亮顯示符合的關鍵字。
  - **批量導入**：在設定頁新增「批量導入」功能，可用 `---SPLIT---` 分隔多條總結，一次解析並導入。
  - **相關對話推薦**：在檢視記錄詳情時，根據「分類、日期、關鍵詞、模式」等，推薦 3 筆最相關的記錄。
  - **Markdown 導出**：新增「導出 Markdown」功能，將所有記錄匯出為結構化的 `.md` 檔案。
- **UI 改良**：新增「回到頂部」按鈕。
- 暂时取消深色模式

---

#### **v4.0 (統計與視覺化儀表板)**
- **核心亮點**：新增「統計」標籤頁，提供數據視覺化儀表板。
- **統計頁面功能**：
  - **頂部數據卡片**：顯示「總記錄數」、「本月記錄數」、「最多分類」。
  - **分類分佈甜甜圈圖**：以圖表呈現各分類的佔比，滑鼠懸停顯示詳細數據。
  - **關鍵詞排行榜 TOP 10**：統計所有記錄中的關鍵詞出現次數，並以進度條視覺化排名。
  - **日曆熱力圖**：以過去 3 個月的日曆方格，直觀顯示每天的記錄數量。
  - **時間趨勢圖 (v4.0 逐步加入)**：
    - `v4.0_no_trend`：僅有日曆熱力圖，無時間趨勢。
    - `v4.0_step2_fixed`：新增「時間趨勢」區塊，可切換 7/14/30 天的每日記錄數長條圖。
- **其他優化**：
  - 響應式設計優化，在平板與桌面上以多欄網格顯示卡片。
  - 儲存鍵名統一為 `ai_dialog_archive_v4`。
  - 暂时取消深色模式

---

#### **v5.0 (儲存架構升級)**
- **核心變革**：導入 **IndexedDB** 作為主要儲存引擎，`localStorage` 降級為備份。
- **新增功能**：
  - **IndexedDB 支援**：解決 `localStorage` 大小限制（~5-10MB）問題，可儲存更多記錄。
  - **自動資料遷移**：首次開啟 v5.0 時，會自動將 v4 的資料從 `localStorage` 遷移至 IndexedDB。
  - **儲存狀態指示器**：頁面頂部顯示「資料庫初始化」、「遷移中」、「就緒」等狀態。
- **版本分支**：
  - `v5.0`：完整版，包含 IndexedDB + localStorage 雙寫與遷移邏輯。
  - `v5.0_LS`：精簡版（僅 localStorage），適用於不希望使用 IndexedDB 的環境。
- **編輯功能 (v5.0 新增)**：在記錄詳情 Modal 中，新增「編輯」按鈕，可修改所有欄位並保存。

---

#### **v5.1 (體驗微調與修正)**
- **主要修正**：基於 v5.0 的架構，進行小幅度修正與體驗優化。
- **推測調整內容**：
  - 修正 IndexedDB 在部分瀏覽器（如 Safari）的兼容性問題。
  - 優化「自動儲存草稿」的體驗與提示。
  - 可能調整了統計頁面的圖表渲染邏輯或 UI 細節。
- **版本定位**：v5.1 是 v5.0 的穩定微調版本，核心功能與儲存架構不變。

---

### 總結演進脈絡

| 版本 | 核心目標 | 關鍵功能 |
| :--- | :--- | :--- |
| **v1.0** | 基礎歸檔 | 儲存、列表、分類、JSON 備份 |
| **v2.0** | 體驗與提醒 | 深色模式、備份提醒、設定頁整合 |
| **v3.0** | 生產力工具 | 全文搜尋、高亮、批量導入、相關推薦、Markdown 導出 |
| **v4.0** | 數據洞察 | 統計儀表板（圖表、熱力圖、趨勢、關鍵詞排行） |
| **v5.0** | 儲存升級 | IndexedDB 支援、自動遷移、記錄編輯功能 |
| **v5.1** | 穩定微調 | 修正 v5.0 的潛在問題，優化使用體驗 |

整體演進方向為：**從「單純儲存」→「智慧管理」→「數據分析」→「可靠儲存」
